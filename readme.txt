Instrucciones

1. Validar la version del JDK compatible con Java 11. Es de desear usar el editor IntelliJ
2. Ejecutar las instrucciones de Maven para la correcta compilacion del aplicativo
   $ mvn clean – borra el directorio de salida (target)
   $ mvn compile – compila el proyecto y deja el resultado en target/classes
   $ mvn package – empaqueta el proyecto y lo dejará en taget/VehicleChallengeAplication-1.0-SNAPSHOT.jar
